### 像数 1, 2, 3 一样容易

``` bash
# install
yarn add # OR npm install

# start
yarn serve # OR npm run serve

# build
yarn build # OR npm run build
```

::: warning 注意
请确保你的 Node.js 版本 >= 8。
:::
