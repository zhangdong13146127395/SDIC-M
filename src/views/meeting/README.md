## 目录结构

```
├─ external                  # 外部会议表单
├─ internal                  # 内部会议表单
├─ meeting-list              # 会议列表
   ├─ apply                  # 我的会议
   ├─ approval               # 我审批的会议 
├─ people                    # 外部人员选择
├─ room                      # 会议室选择
├─ select                    # 会议类型选择
└─ service                   # 服务选择
   ├─ car                    # 车辆选择
   ├─ index                  # 服务选择
```
